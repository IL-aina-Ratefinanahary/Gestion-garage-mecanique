**Gestion de Garage Mécanique**

Description

Ce projet vise à concevoir et développer un ensemble de classes en C++ pour gérer les factures d'un garage mécanique. Le projet met en application des concepts avancés de programmation orientée objet, tels que :

Héritage

Association

Composition

Polymorphisme

Classes abstraites

Délégation

Traitement des exceptions
